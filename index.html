Spring Crypto Utils

Cryptography in Java sucks, badly. That's why we make cryptography tools that are a snap to configure and use.

Start here

Simple

We provide wrappers around the Java Cryptography API so that configuration and use of a bunch of tools can be done the same easy way you are already accustomed to work in your applications using Spring.

Complete

Support for wiring and using all of the stuff cryptography usually requires is built in:

  • key stores
  • public and private keys
  • signers and verifiers
  • symmetric and asymmetric encryption
  • message and mac digesters

Proven

Our customers are online gaming companies that move significant amounts of money 24/7. Also, because the code is open source, you don't need to trust us: you can see for yourself if you like what we do.
